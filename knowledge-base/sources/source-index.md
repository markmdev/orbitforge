## Primary and High-Value Sources

| Source | Why it matters |
|---|---|
| [GAO, Data Centers in Space](https://www.gao.gov/products/gao-26-109012) | Best concise U.S. government overview. Establishes maturity split: smaller in-space data processing is closer than large AI-training datacenters; flags heat, data transfer, cost, and collision risk. |
| [Google Research, Project Suncatcher](https://research.google/blog/exploring-a-space-based-scalable-ai-infrastructure-system-design/) | Serious hyperscaler R&D source for TPUs in space, optical inter-satellite links, and early-2027 Planet prototypes. |
| [Planet, Project Suncatcher platform](https://www.planet.com/pulse/planet-to-build-and-operate-advanced-space-platform-for-project-suncatcher-moonshot/) | Confirms Planet's role building and operating two prototype satellites for Google. |
| [Starcloud-1](https://www.starcloud.com/starcloud-1) | Starcloud's own claim that Starcloud-1 launched in November 2025 with the first NVIDIA H100 in space. |
| [FCC Starcloud public notice](https://docs.fcc.gov/public/attachments/DOC-419509A1.pdf) | Regulatory source for Starcloud's up-to-88,000-satellite distributed datacenter application. |
| [FCC SpaceX public notice](https://docs.fcc.gov/public/attachments/DA-26-113A1.pdf) | Regulatory source for SpaceX's up-to-one-million-satellite ODC filing. |
| [FCC Blue Origin public notice](https://docs.fcc.gov/public/attachments/DOC-420864A1.pdf) | Regulatory source for Blue Origin Project Sunrise, up to 51,600 satellites. |
| [Axiom orbital data center](https://www.axiomspace.com/orbital-data-center) | Near-term ISS/orbital edge compute reference. Useful for cloud-native workloads in space. |
| [Red Hat and Axiom AxDCU-1](https://www.redhat.com/en/about/press-releases/red-hat-teams-axiom-space-launch-optimize-axiom-spaces-data-center-unit-1-orbit) | Software/runtime reference for Red Hat Device Edge, MicroShift, RHEL, and Ansible in orbital edge context. |
| [NVIDIA Space Computing](https://nvidianews.nvidia.com/news/space-computing) | Current accelerator-platform positioning for orbital datacenters, geospatial intelligence, and autonomous space operations. |
| [ASCEND Horizon](https://ascend-horizon.eu/) | EU-funded feasibility-study source for large-capacity data centers in space. |
| [Thales Alenia Space ASCEND results](https://www.thalesaleniaspace.com/en/press-releases/thales-alenia-space-reveals-results-ascend-feasibility-study-space-data-centers-0) | Highlights environmental comparison and the need for much less emissive launchers. |
| [Nature Electronics, carbon-neutral data centres in space](https://www.nature.com/articles/s41928-025-01476-1) | Peer-reviewed framing for orbital edge and orbital cloud datacenter frameworks plus lifecycle carbon comparison. |
| [DOE/LBNL data center energy report release](https://www.energy.gov/articles/doe-releases-new-report-evaluating-increase-electricity-demand-data-centers) | Terrestrial energy baseline: 2023 U.S. data center electricity and 2028 projections. |
| [ESA Space Environment Report](https://www.sdo.esoc.esa.int/environment_report/Space_Environment_Report_latest.pdf) | Debris and orbital environment risk baseline. |
| [ESA Space Environment Statistics](https://sdup.esoc.esa.int/discosweb/statistics/) | Current debris statistics page, updated June 2026. |
