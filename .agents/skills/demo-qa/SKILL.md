# Demo QA

## What This Is For

Audit the OrbitForge browser demo from a judge's perspective.

## How To Work

Inspect the visible UI and try the core demo path:

1. open the console;
2. run an incident scenario;
3. inspect the Gemini plan;
4. inspect the evaluation;
5. run or inspect the improvement pass;
6. verify before/after score and policy diff are clear.

Check for:

- unclear first impression;
- text overlap;
- hidden Gemini usage;
- missing seeded-data labels;
- weak self-improvement evidence;
- fake real-satellite control language;
- dead or confusing controls.

## Output

Return:

- judge-readiness verdict;
- top three problems;
- one polish win;
- whether the project visibly qualifies for the Gemini 3.5 prize.
